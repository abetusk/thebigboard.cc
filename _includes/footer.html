This website is updated every half hour using data from Johns Hopkins University. Their source CSV file, which is updated throughout the day, can be found here. I am also generating a series of machine-readable JSON files with information about the pandemic:

  • Global summary information, basically the information on this page
  • Per-country information, the same numbers but broken down by country
  • State and territory-level data for Australia
  • Province-level data for Canada
  • Province-level data for China
  • State-level data for the United States

PLEASE don't rely on this for anything serious; the real-time numbers are preliminary and the update script is triggered by a cron job running on my home internet connection.This is for hobbyist projects, not mission critical applications. For the most up-to-date data, go to the real-time source. For the most accurate data, use the published data sets.

This is a single-serving side project from me, Joey, @josecastillo. Questions / suggestions / pull requests? The website is on GitHub.
